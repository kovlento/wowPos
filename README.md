## wowpos

### 一、简介
基于vue2.0实现的简单的点餐系统(只实现了一部分功能...)，技术栈 vue2.0 +  webpack +  es6 + axios；基本效果见下面：

![首页](/img/index.png)

### 二、如何执行

####  1、将项目克隆到本地
```bash
git clone git@github.com:kovlento/wowPos.git
```
#### 2、安装依赖
```bash
npm install
```
#### 3、执行
```bash
npm run dev
```
#### 4、打开浏览器浏览 http://localhost:8080/
